## How `/leadership-engine` uses these

When generating Monday Morning Brief:

1. Reads BUSINESS-BRAIN.md § Voice / Identity
2. Displays today's identity reminder (rotated)
3. Shows current quarter's WIGs + Rocks status
4. Surfaces calendar block analysis (maker % / manager %)
5. Suggests today's PRIORITY action

When generating Sunday Reflection:

1. Audits past week against current Rocks
2. Scores week's maker time vs target
3. Surfaces drift (work that's NOT a Rock or whirlwind)
4. Generates Monday's L10 prep

When running Quarterly Review:

1. 4-decisions audit (People / Strategy / Execution / Cash)
2. WIG completion scoring
3. V/TO update walkthrough
4. 17-pattern wealth-mindset audit
5. 6-faculty audit
6. 10X dimension review
7. Saves quarterly review document

## The leadership operating cycle (combine all 8)

```
DAILY     ·  identity reminder · 1-question diagnostic
WEEKLY    ·  re-read V/TO · L10 meeting · maker time review
MONTHLY   ·  WIG + Rocks audit · maker calendar review
QUARTERLY ·  4 decisions audit · 17-pattern audit · 10X dimension review
ANNUAL    ·  V/TO refresh · wealth-mindset full audit · re-pick the
              focus quarter pattern · re-do delete list
```
