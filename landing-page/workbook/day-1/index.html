3.1c · Manual setup (if .ics doesn't open in your calendar)

Click here for step-by-step manual setup · Google Calendar / Apple Calendar / Outlook

Google Calendar

  1. Open calendar.google.com
  2. Settings (gear icon) → Settings → Import & export → Import
  3. Select the downloaded pp-week.ics file → click Import
  4. All 6 events appear · already set to repeat weekly

Apple Calendar (Mac)

  1. Double-click the downloaded pp-week.ics file · Calendar opens automatically
  2. Click Add All when the dialog appears
  3. Pick which calendar (Personal / Work / iCloud) · click OK

Outlook

  1. Outlook desktop · File → Open & Export → Import/Export
  2. Choose Import an iCalendar (.ics) or vCalendar file → Next
  3. Browse to pp-week.ics · click Import

If you'd rather create them by hand · 6 calendar entries

For each row in the table above:

  1. Title · paste the slash command (e.g. /purely-personal:marketing-engine)
  2. Day + Time · as shown in the table
  3. Duration · 15 minutes
  4. Repeat · "Every week on [day]" · indefinitely
  5. Notification · 5 minutes before
  6. Save
