# Rewrite Standards
# Tailor Reference, Purely Personal Bootcamp

Skill-specific rewrite instructions for each of the 5 starter skills.
Read the relevant section before rewriting each skill.

---

## UNIVERSAL REWRITE RULES (apply to all skills)

**Replace every generic placeholder:**
- "your ICP" → [ICP ROLE] at [COMPANY STAGE] who [SITUATION]
- "your offer" → [OFFER NAME]
- "your audience" → [specific description]
- "your competitors" → [COMPETITOR 1], [COMPETITOR 2], [COMPETITOR 3] by name
- "Daniel Paul" → [PARTICIPANT NAME] everywhere it appears as the voice default

**Inject specificity at every section:**
If you removed the participant's name, would it still be clearly theirs?
If yes, not specific enough.

**Never break the workflow logic:**
Add context into steps. Never reorder, combine, or remove steps.

**Never invent proof points:**
If none listed, write "[ADD PROOF POINT: result + timeframe + client type]" as placeholder.

## SKILL 1, CONTENT STRATEGY REWRITE

**STEP 1, INTAKE:** Remove questions 1-2 (already known). Keep 3-7 but reframe around participant context.

**STEP 2, DEEP RESEARCH:** Search queries must name the specific niche:
- Generic: "[niche] LinkedIn creators"
- Fitted: "[INDUSTRY + ICP ROLE] LinkedIn content [year]"
Add 5th search: "[PARTICIPANT NAME] competitors positioning [year]"

**STEP 3, POSITIONING STATEMENT:** Must name specific ICP role and company stage, reference participant's UNIQUE ANGLE, explicitly state what top 2 competitors are known for then position against it.

**STEP 5, CONTENT PILLARS:** Each pillar must be unownable by any competitor. Ask: could [competitor name] use this pillar? If yes, rewrite. Connect to specific ICP pain. Reference participant's real proof points.

**STEP 6, COMPETITOR CONTENT STUDY:** Replace generic slots with actual top 3-5 from Tailor Briefing. Each gap section must name specific topics they avoid or handle poorly.

## SKILL 2, LINKEDIN CAPTION WRITER REWRITE

**VOICE LOCK (replaces Voice DNA Check):** Extract all 8 voice dimensions and embed directly. Not a reference, embedded verbatim. Minimum: top 5 rules (specific), banned words list (minimum 10), sentence rhythm, CTA format example, hook archetype preference.

**STEP 2, INTAKE:** Pre-fill participant name, ICP, offer. Only ask for: specific idea/topic/result for this post + any new proof point.

**STEP 5, HOOK SELECTION:** Add participant's preferred hook archetypes from Voice DNA.

**STEP 6, WRITE BODY:** Add specificity standard: "Every post must contain at least one detail that could only come from [PARTICIPANT]."

**STEP 8, INVISIBILITY DIAGNOSTIC:** Add check: "Voice: sounds like [PARTICIPANT], not like [COMPETITOR 1] or [COMPETITOR 2]."

**Common mistakes:** Keeping "write in Daniel Paul's voice." Leaving intake asking for pre-known info. Voice Lock rules too generic ("be specific", "be direct").

## SKILL 3, DM SEQUENCE WRITER REWRITE

**VOICE LOCK:** Same as Caption Writer. Add: "DMs are shorter and more direct than posts. Energy level drops to [N-1] from standard post level."

**STEP 2, INTAKE:** Pre-fill offer, ICP. Only ask per session: prospect details and temperature.

**MESSAGE 1:** Add participant's typical human hooks: "[PARTICIPANT]'s strongest connection hooks come from: [top 2-3 signal types]."

**MESSAGE 3:** Add participant's specific value delivers: "[PARTICIPANT]'s strongest value bridges: [2-3 specific insights or frameworks]."

**CARDINAL RULE:** "If you could send this message as [PARTICIPANT] to someone else entirely, it fails."

## SKILL 4, NEWSLETTER WRITER REWRITE

**STEP 1, INTAKE:** Pre-fill participant, ICP, offer. Only ask per session: this week's topic/lesson, win, takeaway, one ask.

**THIS WEEK'S WIN:** Hard rule: "Must use a real number. Standing proof points: [list from Tailor Briefing]. Never use vague language here."

**SECTION 3, ONE FRAMEWORK:** "Name must reference [PARTICIPANT]'s methodology. '[PARTICIPANT]'s [Framework Name]' is stronger than a generic name."

## SKILL 5, SALES CALL PREP REWRITE

**STEP 1, INTAKE:** Pre-fill participant name, offer name, transformation, ICP profile.

**SECTION 4, QUALIFICATION QUESTIONS:** Add ICP-specific pain questions before generic BANT+.

**SECTION 6, OBJECTION SCRIPTS:** Replace generic objections with 3 most likely for THIS participant's ICP and offer. Every script: Stated / Real / Response (in participant's voice) / Move.

**SECTION 7, TRANSITION TO CLOSE:** "Based on everything you've shared, I think there's a real fit for [OFFER NAME]."

**SECTION 9, POST-CALL FOLLOW-UP:** Update sign-off from "Daniel" to [PARTICIPANT NAME].

**Common mistakes:** Leaving "Daniel" as sign-off. Using generic objection scripts. Not updating offer name in close transition.
