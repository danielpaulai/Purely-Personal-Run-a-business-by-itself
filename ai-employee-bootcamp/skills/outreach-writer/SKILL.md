# ◈ THE OUTREACH WRITER

> Templates are what people send when they don't care who they're talking to. This skill writes the opposite.

Every message in this sequence is built from scratch, around this specific person, this specific moment, and the voice of the human sending it. The goal is not to sound good. The goal is to sound real.

---

## ━━ THE CARDINAL RULE ━━

> "If you could send it to anyone, send it to no one."

Every message must pass this test before it leaves this skill. Generic is invisible. Specific is magnetic.

---

## ━━ BEFORE YOU WRITE ━━

### Step 1, Get the Intelligence Brief
If The Prospector has been run, pull the full brief. If not, collect:
- Prospect name, role, and industry
- Conversation temperature (Cold / Warm / Hot)
- Communication style (Driver / Expressive / Amiable / Analytical)
- The Human Hook (10–14 words, specific, verified)
- Recommended opener type
- The prospect's most likely pain point

### Step 2, Profile Audit
The strongest sequence dies on a weak profile. Before writing a single word, audit the sender's LinkedIn presence across three questions:

| Audit Point | Green light | Red flag |
|---|---|---|
| **Headline** | Names who you help and the result you deliver | Generic job title with no benefit |
| **About section** | Opens with the reader's problem, not your credentials | Starts with "I am a..." |
| **Social proof** | Visible recommendations, case studies, or results | Empty or purely self-promotional |

**Issue the verdict:**
```
┌─────────────────────────────────────────────┐
│  PROFILE AUDIT                              │
├─────────────────────────────────────────────┤
│  Headline:      ✓ PASS  /  ⚠ WARNING        │
│  About:         ✓ PASS  /  ⚠ WARNING        │
│  Social Proof:  ✓ PASS  /  ⚠ WARNING        │
├─────────────────────────────────────────────┤
│  OVERALL: ✓ PROCEED / ⚠ FIX FIRST          │
└─────────────────────────────────────────────┘

[If WARNING or FIX FIRST, name the exact issue and what to change.
One sentence per issue, no vague advice.]
```

If the user skips the audit, flag it at the end of the sequence and proceed.

### Step 3, Capture Voice DNA
The sequence must sound like the person sending it, not like Claude, not like a sales coach, not like anyone else.

Ask: *"Paste a sample of how you normally write, a recent message, a post, or just describe your style in 3 words."*

Extract and note:
- **Sentence length:** short and punchy / longer and flowing
- **Punctuation style:** minimal / expressive / formal
- **Energy level:** high warmth / cool confidence / dry wit / direct
- **Signature phrases or patterns:** anything they say that no one else would

## ━━ THE 5-MESSAGE SEQUENCE ━━

Write all five messages before delivering. Each one must be reviewed against the quality gate at the end.

---

### MESSAGE 1, CONNECTION REQUEST
**Hard limit: 300 characters. No exceptions.**

This message has one job: get accepted. Not impress. Not pitch. Not qualify. Get accepted.

**The formula:**
- One genuine observation that proves you looked (not the Human Hook, save that for M2)
- A soft signal of relevance (shared niche, their content, a mutual context)
- Sound like a peer making a natural connection, not a vendor prospecting

**Structural rules:**
- No "I'd love to connect" (passive and generic)
- No pitch, no offer, no mention of what you do
- No question that requires effort to answer
- Read aloud, if it sounds like an email, cut it in half

**Output:**
```
┌─────────────────────────────────────────────┐
│  MESSAGE 1, CONNECTION REQUEST             │
│  [X / 300 characters]                       │
├─────────────────────────────────────────────┤
│  [Message text]                             │
└─────────────────────────────────────────────┘
```

---

### MESSAGE 2, THE ICEBREAKER
**Timing: Send immediately after connection accepted**

This is the most important message in the sequence. It sets the entire tone of the relationship. The Human Hook lives here.

**Universal rules:**
- The Human Hook must be embedded naturally, not bolted on
- Match the communication style in every word choice and sentence length
- Energy is 4/10, curious and human, never eager or salesy
- Zero mention of your offer
- One open question at the end
- Under 5 sentences total

**Select the opener based on the Intelligence Brief:**

**Peer Opener**, Warm prospects, Expressive/Amiable styles:
> *"Great to connect [NAME]. No agenda on my end lol.*
> *Quick one while we're new to each other, are you more of a content scroller or an actual conversationalist on here?*
> *Either works :)*
> *P.s. [Human Hook]"*

**Pattern Interrupt**, Expressive types, competitive niches:
> *"Hey [NAME], good to be connected.*
> *I talk to a lot of [industry] people, most are grinding with [pain point].*
> *You seem to be taking a different angle. What's actually working for you right now?"*

**Directness Play**, Driver/Analytical types, cold temperature:
> *"Hey [NAME], I'll skip the formalities.*
> *Noticed something on your [profile/content] that I think I could help with, quickly and at no cost.*
> *Happy for me to share?"*

**Authority Flip**, High-status prospects, strong profiles:
> *"Hey [NAME], I don't send many of these.*
> *Your [specific thing] stopped me when I was looking through [niche].*
> *[One genuine question tied to what stood out]"*

**Output:**
```
┌─────────────────────────────────────────────┐
│  MESSAGE 2, ICEBREAKER                     │
│  Opener style: [type]                       │
├─────────────────────────────────────────────┤
│  [Message text]                             │
└─────────────────────────────────────────────┘
```

---

### MESSAGE 3, VALUE-ADD FOLLOW-UP
**Timing: 2–3 days after M2 if no reply**

This is not a chase. This is a return with something worth reading.

**The principle:** If the first message opened a door, this one puts something interesting on the doorstep. They find it when they come back.

**Structure:**
- Don't reference being ignored, it frames you as needy
- Bring a genuine piece of value: a pattern you've noticed in their niche, a specific observation about their content or market, a question their last post made you think about
- One soft question to invite a response
- Under 4 sentences

**Output:**
```
┌─────────────────────────────────────────────┐
│  MESSAGE 3, VALUE-ADD FOLLOW-UP            │
│  Timing: 2–3 days after M2 if no reply      │
├─────────────────────────────────────────────┤
│  [Message text]                             │
└─────────────────────────────────────────────┘
```

---

### MESSAGE 4, THE PIVOT
**Timing: 3–4 days after M3 if no reply**

Come from a completely different angle. Same conversation, fresh entry point.

**The principle:** If two messages from the same direction didn't land, change direction entirely. Not in desperation, in genuine curiosity. Ask about something different. Reference something new. Let this feel like the first message of a different conversation.

**Rules:**
- Do not reference previous messages
- Do not repeat any observation or question from M2 or M3
- Under 3 sentences
- End with an easy yes/no or a very low-effort question

**Output:**
```
┌─────────────────────────────────────────────┐
│  MESSAGE 4, THE PIVOT                      │
│  Timing: 3–4 days after M3 if no reply      │
├─────────────────────────────────────────────┤
│  [Message text]                             │
└─────────────────────────────────────────────┘
```

---

### MESSAGE 5, THE CLEAN EXIT
**Timing: 5–7 days after M4 if no reply**

Most people either keep chasing or disappear without a word. Both are wrong. This message is the third option, and it's the one that leaves the door permanently open.

**The principle:** Close the loop with warmth and no resentment. This is not passive-aggressive. This is high-status. You're not desperate. You're simply closing the sequence with respect for both of you.

**Rules:**
- Acknowledge that you're stopping, don't ghost
- No guilt-trip, no guilt-bait ("I guess you're not interested...")
- Leave a genuine door open without expectation
- Under 3 sentences
- Walk away with your dignity fully intact

**Output:**
```
┌─────────────────────────────────────────────┐
│  MESSAGE 5, CLEAN EXIT                     │
│  Timing: 5–7 days after M4 if no reply      │
├─────────────────────────────────────────────┤
│  [Message text]                             │
└─────────────────────────────────────────────┘
```

## ━━ QUALITY GATE ━━

Run every message through this before delivering. If any answer is No, rewrite:

```
[ ] Could this message have been sent to someone else? → If yes, it's too generic
[ ] Does it sound like the sender, not like a sales script? → Voice DNA match
[ ] Is there any hint of pitch or qualification? → Remove it (M1–M4 only)
[ ] Does it end with an invitation to reply? → Every message needs one
[ ] Is the energy calm and peer-to-peer? → Not eager, not cold
[ ] Is the Human Hook embedded naturally in M2? → Not forced, not bolted on
[ ] Would the sender be proud to send this? → The final test
```

---

## ━━ NON-NEGOTIABLE RULES ━━

- **The Human Hook must appear in M2.** Not summarised. Quoted exactly as written in the Intelligence Brief.
- **No pitch before M6+.** The sequence is for relationship-building. Selling comes in The Closer.
- **One question per message.** Stacking questions is anxious energy. One question, asked well, is confident.
- **Read every message aloud.** If it sounds like it was written by software, it was. Rewrite it.
- **Voice DNA first, technique second.** A technically perfect message in the wrong voice kills the deal.
