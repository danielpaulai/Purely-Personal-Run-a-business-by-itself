# Install steps (read these to the user verbatim)

Your new employee is a small folder. Here is how to put it to work on your own Claude account.

## Save the folder
1. Download the zip I made for you. Inside is the employee folder: the `SKILL.md` plus a `references` folder with the files the skill needs (for example `scrub.md`, and `design-system.md` if it makes branded visuals).
2. Keep them together. The skill will not work right if you pull the `SKILL.md` out on its own, because it reads those reference files. This is why we hand you a zip, not a single file.

## Install it in Claude
1. Open Claude in your browser at claude.ai.
2. Click your name or the settings area, then go to **Customize**.
3. Open **Skills**.
4. Click the **+** button, then **Upload a Skill**.
5. Drag and drop the zip in. Do not unzip it first. Claude reads the whole folder, the skill and its references together.
6. Wait for it to show up in your skills list. Done. Your employee is hired.

## Wake it up
- Start a new chat. Type the trigger phrase, for example "write me a proposal for ____."
- Or type a forward slash and the employee's name to call it directly.
- Give it one real task right now to test it. Watch what it hands back. If something is off, tell me and we will adjust the skill.

## One reminder
Each person runs the skill on their own Claude account. If you want a team member or a client to use it, share the file and have them install it on their account. Do not run it on your account pretending to be them, it will mix up the context and the output will drift.
